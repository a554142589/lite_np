# litenp Documentation

These are the canonical public docs for the current `litenp` release candidate.

- `api_compatibility.md`: supported NumPy/libtorch-like semantics and explicit
  differences.
- `benchmark_methodology.md`: how benchmark rows are categorized and how to read
  speedups.
- `benchmark_2026-05-27.md`: latest benchmark report.
- `views_and_lifetime.md`: `ArrayView<T>` ownership and mutation notes.
- `design.md`: high-level design notes.

This directory intentionally contains only public user-facing docs.
